Instalation:
--------------------------------------------------------
Download as zip and place into your resourcepack folder.

Required mods to function:
--------------------------------------------------------
ANIMATICA, ENTITY TEXTURE FEATURES, CUSTOM ENTITY MODEL, CLOTH CONFIG API, COMPLETE CONFIG
FABRIC LOADER, FABRIC API
